## 🛠️ Como executar os projetos

1. Clone este repositório:
   ```bash
   git clone https://github.com
   ```
2. Navegue até a pasta do projeto desejado:
   ```bash
   cd nome-da-pasta
   ```
3. Execute o script:
   ```bash
   python nome-do-arquivo.py
   ```
