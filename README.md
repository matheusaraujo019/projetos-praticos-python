# 🐍 Meus Projetos Práticos de Python

Bem-vindo ao meu repositório de estudos! Aqui organizo pequenos scripts e ferramentas que desenvolvi para consolidar meus conhecimentos em Python, lógica de programação e automação.

---

## 📑 Índice de Projetos

Abaixo, você encontrará uma lista de cada projeto contido neste repositório, com uma breve explicação do que ele faz e o que eu pratiquei nele.

### 1. 🧮 Calculadora de IMC
*   **Pasta:** `/calculadora-imc`
*   **O que faz:** Calcula o Índice de Massa Corporal com base no peso e altura inseridos.
*   **Conceitos aplicados:** Entrada de dados, condicionais (if/else) e tratamento de erros.

### 2. 🔑 Gerador de Senhas Seguras
*   **Pasta:** `/gerador-senhas`
*   **O que faz:** Cria senhas aleatórias com letras, números e símbolos.
*   **Conceitos aplicados:** Uso da biblioteca `random` e manipulação de strings.

### 3. 🌐 Web Scraper Simples
*   **Pasta:** `/web-scraper`
*   **O que faz:** Coleta o preço de um produto em um site específico.
*   **Conceitos aplicados:** Bibliotecas `requests` e `BeautifulSoup`.

---

## 🛠️ Como executar os projetos

1. Clone este repositório:
   ```bash
   git clone https://github.com
   ```
2. Navegue até a pasta do projeto desejado:
   ```bash
   cd nome-da-pasta
   ```
3. Execute o script:
   ```bash
   python nome-do-arquivo.py
   ```
